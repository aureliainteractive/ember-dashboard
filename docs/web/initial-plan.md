- Dispositivos conectados a la red
- Control de la Cabina mediante endpoint directo a ember-server (Revisar ember-server-readme.md)

- Detalles de sesión actual.
- Verificación de Servicios hacia conexión de servidores individuales: roblox.com cdn.roblox.com cloudflare.com api.aureliainteractive.me
